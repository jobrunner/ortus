# ortus documentation

ortus is a Go REST service for **point queries on GeoPackage and raster
sources** — find the geographic features (or raster values) that contain a
coordinate, across many sources at once, via SpatiaLite.

This documentation follows the [Diátaxis](https://diataxis.fr/) framework — four
kinds of docs for four kinds of need:

| | |
|---|---|
| **[Tutorials](tutorials/index.md)** | Learning-oriented — start here for your first run. |
| **[How-to guides](how-to/index.md)** | Task-oriented — recipes for a specific goal (deploy, enable TLS, sync, load-test). |
| **[Reference](reference/index.md)** | Information-oriented — config keys, the HTTP API, MCP tools, observability. |
| **[Explanation](explanation/index.md)** | Understanding-oriented — architecture, decisions (ADRs), the tech-debt policy. |
